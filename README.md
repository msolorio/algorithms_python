## Run the tests

Create virtual env, activate virtual env, and install requirements
```sh
python3 -m venv .env
source .env/bin/activate
pip3 install --upgrade pip
pip3 install -r requirements.txt
```

Run tests
```sh
pytest
```

Deactivate venv
```sh
deactivate
```
